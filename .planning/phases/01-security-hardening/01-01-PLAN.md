    2. **_FORBIDDEN_PATTERNS list** (regex patterns to reject before sympify):
       ```python
       _FORBIDDEN_PATTERNS = [
           r'__',          # dunder access (__import__, __class__, etc.)
           r'\bimport\b',  # import statements
           r'\beval\b',    # eval/exec
           r'\bexec\b',
           r'\bopen\b',    # file access
           r'\bos\b',      # os module
           r'\bsys\b',     # sys module
           r'\bgetattr\b', # attribute introspection
           r'\bsetattr\b',
           r'\bglobals\b',
           r'\blocals\b',
           r'\bcompile\b',
           r'\bbreakpoint\b',
           r'\blambda\b',  # no lambda definitions
           r'\bclass\b',
           r'\bdef\b',
           r'\bfor\b',
           r'\bwhile\b',
           r'\breturn\b',
           r'\byield\b',
           r'\braise\b',
           r'\btry\b',
           r'\bwith\b',
           r'\bdel\b',
           r'\bprint\b',
       ]
       ```

    3. **_ALLOWED_CHARS regex** for standard math expressions:
       ```python
       _ALLOWED_CHARS = re.compile(r'^[a-zA-Z0-9\s\+\-\*/\(\)\[\]\.\,\^\_]+$')
       ```

    4. **_ALLOWED_CHARS_WITH_COMPARISONS regex** for condition expressions (adds < > = !):
       ```python
       _ALLOWED_CHARS_WITH_COMPARISONS = re.compile(r'^[a-zA-Z0-9\s\+\-\*/\(\)\[\]\.\,\^\_\<\>\=\!]+$')
       ```

    5. **_SYMPY_MATH dict** mapping function names to sympy equivalents:
       ```python
       _SYMPY_MATH = {
           'sin': sympy.sin, 'cos': sympy.cos, 'tan': sympy.tan,
           'asin': sympy.asin, 'acos': sympy.acos, 'atan': sympy.atan,
           'atan2': sympy.atan2, 'arctan2': sympy.atan2,
           'exp': sympy.exp, 'log': sympy.log,
           'log10': lambda x: sympy.log(x, 10),
           'sqrt': sympy.sqrt, 'abs': sympy.Abs,
           'pi': sympy.pi, 'e': sympy.E,
           'sinh': sympy.sinh, 'cosh': sympy.cosh, 'tanh': sympy.tanh,
           'sign': sympy.sign, 'floor': sympy.floor, 'ceil': sympy.ceiling,
           'heaviside': sympy.Heaviside,
       }
       ```

    6. **validate_math_input(expr_str: str) -> str:**
       - Strip whitespace, reject empty
       - Check each _FORBIDDEN_PATTERNS with re.search, raise ValueError if match
       - Check _ALLOWED_CHARS.match, raise ValueError if no match
       - Return stripped string

    7. **validate_math_input_with_comparisons(expr_str: str) -> str:**
       - Same as validate_math_input but uses _ALLOWED_CHARS_WITH_COMPARISONS

    8. **safe_build_ode_func(function_str: str, num_components: int) -> callable:**
       - Call validate_math_input(function_str)
       - Create IndexedBase('y') and symbols('t')
       - Build local_dict = dict(_SYMPY_MATH, y=y, t=t)
       - Parse with sympify(validated, locals=local_dict)
       - Handle tuple result (multi-component) vs single expression
       - Substitute y[i] -> plain symbols _y0, _y1, etc. for lambdify compatibility
       - lambdify each component with modules='numpy'
       - Return closure: def ode_func(t_val, y_val): return np.array([f(t_val, *y_val) for f in funcs])

    9. **safe_build_1d_func(function_str: str) -> callable:**
       - validate_math_input, sympify with x=symbols('x') in local_dict
       - lambdify(x, parsed, modules='numpy')

    10. **safe_build_multi_func(function_str: str, var_names: list) -> callable:**
        - validate_math_input, create symbols for each var_name
        - sympify with all vars in local_dict
        - lambdify with ordered symbol list

    11. **safe_build_condition_func(condition_str: str, var_names: list) -> callable:**
        - validate_math_input_with_comparisons (allows < > = !)
        - sympify produces SymPy relational object
        - lambdify with modules='numpy'
