## Interaction Contracts

### SectionAccordion

- **Default state:** All sections collapsed after generation. Each shows section name (Hebrew or English per language toggle) + AIGeneratedBadge if unedited.
- **Expand:** Click accordion header. Shows rendered KaTeX preview of section content. "Edit" button visible at top-right of expanded panel.
- **Edit mode:** Click "Edit" replaces single-column preview with side-by-side: textarea (left/right based on language) + live KaTeX preview. Preview updates on every keystroke with 300ms debounce.
- **Save/close:** Click "Done Editing" collapses back to preview-only. No explicit save -- text state lives in React useState.
- **Badge clearing:** AIGeneratedBadge disappears on first character change in textarea. Does not reappear.

### TitlePageForm

- **Placement:** Above the section accordions, below the "Generate Report" button area.
- **Fields:** Student Name (text), Student ID (text), Lab Partner (text), Course Name (text), Experiment Title (text, pre-filled from context form if available), Date (date input, defaults to today).
- **Labels:** Hebrew by default, follows language toggle.
- **Validation:** Name and Experiment Title required for PDF export. Red border + inline error text if empty on export attempt.

### TemplateSelector

- **Placement:** Inline with Export PDF button, to its left.
- **Options:** "Israeli Lab Report" (default), "Minimal Clean", "LaTeX-Inspired Academic".
- **Behavior:** Native `<select>` element styled to match input conventions. Selection persists in React state.

### ExportPDFButton

- **Placement:** Fixed at bottom of report page, full-width within the 700px max-width container.
- **Idle:** Blue gradient background (matching existing Generate Report button style). Label: "Export as PDF".
- **Exporting:** Disabled, gray background, spinner icon, label: "Generating PDF...".
- **Success:** Browser download triggered. Button returns to idle.
- **Error:** Red text below button with error message + "Try Again" link.
- **Gate:** Disabled if no generated sections exist. Disabled if required title page fields are empty (name, experiment title).

### PlotThumbnail

- **Source:** Plotly.toImage() called on fit plot and residuals plot from AutoLab results in AnalysisContext.
- **Display:** 100% width of accordion content area, max-height 300px, object-fit contain.
- **Missing plots:** Gray placeholder box with text "No plot available -- run AutoLab analysis first".
- **Caption format:** "Figure 1: Fit Plot -- {model_name}" / "Figure 2: Residuals".

## PDF Template Contract

### Israeli Lab Report (default, per D-04)

| Property | Value |
|----------|-------|
| Page size | A4 (210mm x 297mm) |
| Margins | 25mm all sides |
| Direction | RTL for Hebrew, LTR equations inline |
| Body font | Noto Sans Hebrew, 12pt, weight 400 |
| Section headings | Noto Sans Hebrew, 14pt, weight 700 |
| Title page | University-style: experiment title centered, student details below, date at bottom |
| Section numbering | Arabic numerals: 1. 2. 3. etc. |
| Figure captions | Centered below figure, 10pt, weight 400, format: "איור {N}: {description}" |
| Parameter table | Bordered table, header row bold, values use roundWithUncertainty format |
| Page numbers | Bottom center, Arabic numerals |
| Header | None (clean academic style) |
| Footer | Page number only |

### Minimal Clean

| Property | Value |
|----------|-------|
| Inherits Israeli template with: thinner margins (20mm), no title page (title as first-page header), lighter section dividers (0.5px border-bottom instead of bold text) |

### LaTeX-Inspired Academic

| Property | Value |
|----------|-------|
| Inherits Israeli template with: Computer Modern-style serif feel via KaTeX_Main font for body where available, two-line header with experiment title + date, formal abstract-style formatting for conclusions |
