## Page Layout: Unified AutoLab Progressive Flow

### Stage 1: Upload + Instructions (existing, minor changes)

```
+--------------------------------------------------+
|  Upload Section (card)                            |
|  [File upload area] [Instructions textarea]       |
|  [Theo value] [Theo uncertainty]                  |
|  [ Try Free Fall Demo ]  ---- [ Run Analysis ]    |
+--------------------------------------------------+
```

- File upload, instructions, theoretical value fields remain as-is
- Example datasets reduced to single "Free Fall Demo" button (btn-secondary)
- "Run Analysis" remains the primary CTA (btn-primary gradient)

### Stage 2: Analysis Results (existing, inline chat removed)

```
+--------------------------------------------------+
|  Analysis Summary (success box, green border)     |
|  AI-generated 3-5 sentence summary                |
+--------------------------------------------------+
|  Parameter Table                                  |
|  | Param | Rounded | Full | Latex |              |

+--------------------------------------------------+

+--------------------------------------------------+
|  N-Sigma Comparison (color-coded)                 |
+--------------------------------------------------+
|  Fit Plot          |  Residuals Plot              |
+--------------------------------------------------+
|  PDF Export: [ Export Results PDF ]                |
+--------------------------------------------------+
```

- Inline AI chat panel removed entirely
- Results-only PDF export button added at bottom of results section
- All other result elements unchanged

### Stage 3: Report Expander (new)

```
+--------------------------------------------------+
|  [ Generate Full Lab Report  v ]                  |
|  Expand your analysis into a complete academic    |
```

- Collapsed by default after analysis completes
- Prominent CTA button (btn-accent gradient, full width, 48px height)
- Subtitle text below button in `--text-secondary` color
- Chevron icon rotates 180deg on expand (CSS transform)
- Expand animation: `max-height` from 0 to content height, 300ms ease-out

## Interaction States

### Analysis Flow

| State | Visual | User Action |
|-------|--------|-------------|
| Idle | Upload area visible, no results | Upload file + enter instructions |
| Analyzing | Spinner + "Analyzing..." text on disabled Run button | Wait |
| Results shown | Summary + params + plots visible, Report Expander appears below | Review results, optionally expand report |
| Analysis error | Red error box with message + "Try adjusting your instructions" help text | Edit instructions, retry |

### Report Flow

| State | Visual | User Action |
|-------|--------|-------------|
| Collapsed | "Generate Full Lab Report" button visible | Click to expand |
| Expanded, empty | Context form visible, Generate button enabled | Fill form, click Generate |
| Generating | Spinner on Generate button, "Analyzing context..." then "Generating sections..." | Wait |
| Follow-up | Question cards appear between form and sections | Answer questions, submit |
| Sections ready | Accordion with 5 sections, each expandable with editor | Review/edit sections |
| Exporting PDF | "Generating PDF..." on disabled export button | Wait |
| PDF ready | Browser download triggered | Save file |
| PDF error | Red error box: "PDF generation failed. {error detail}. Try again or contact support." | Retry |

### Report Expander Animation

- **Expand:** Button chevron rotates 0deg to 180deg over 200ms. Report section `max-height` animates from 0 to `scrollHeight` over 300ms ease-out. Opacity fades from 0 to 1 over 200ms (delayed 100ms).
- **Collapse:** Reverse of expand. Content fades out first (200ms), then height collapses (300ms ease-in).
- **Scroll behavior:** On expand, smooth-scroll the report section into view after animation completes (400ms delay).

## PDF Export Presentation

Two buttons in a horizontal button group at bottom of results section and bottom of report section:

```
[ Export Results PDF ]  [ Export Full Report PDF ]
     btn-secondary            btn-accent
     always enabled       disabled until sections exist
```

- Results PDF: Exports analysis summary, parameter table, plots only. No AI sections, no title page.
- Full Report PDF: Exports complete academic report with all sections, title page, plots, formatted per template selection.
- Both buttons show inline spinner + "Generating PDF..." when clicked.
- On success: browser download triggers, button returns to normal.
- On failure: red text appears below buttons with error message.
